# OWASP Juice Shop — 6-Star Challenge Write-ups

This repository contains my personal write-ups for all **6-star (⭐⭐⭐⭐⭐⭐)** challenges in the [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/), an intentionally vulnerable web application designed for security training and penetration testing practice.

6-star challenges represent the highest difficulty tier in Juice Shop — expect advanced exploitation techniques, chained vulnerabilities, and a fair amount of creative thinking.

Each write-up is available in both Markdown and PDF format.

## About OWASP Juice Shop

OWASP Juice Shop is widely regarded as one of the most comprehensive intentionally insecure web applications available for security training. It covers vulnerabilities from the OWASP Top 10, OWASP API Security Top 10, and beyond — all gamified through a score board and challenge system.

More info: [https://owasp-juice.shop](https://owasp-juice.shop)
